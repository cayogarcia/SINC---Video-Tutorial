# 🎥 SINC - Vídeo Tutorial

Sistema de gerenciamento e exibição de vídeos tutoriais com controle de acesso por tipo de usuário.

## 📌 Tecnologias utilizadas

### 🔹 Frontend
- React
- TypeScript
- Context API
- Axios

### 🔹 Backend
- FastAPI (Python)
- MariaDB
- SQLAlchemy
- JWT Authentication
- Alembic (migrations)

### 🔹 Outros
- Docker (parcial)
- Git e GitHub
